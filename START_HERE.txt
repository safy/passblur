Перед стримом:
  1. Откройте PassBlur popup (клик на иконку)
  2. Убедитесь что защита включена (зеленый переключатель)
  3. Откройте страницы с кодом, которые будете показывать
  4. Проверьте счетчик защищенных ключей
  5. Начинайте стрим!

Во время стрима:
  → Открыли новые файлы? Нажмите "Пересканировать"
  → Нужно скопировать ключ? Просто кликните по нему
  → Хотите временно показать ключ? Выключите защиту в popup

⚠️ ВАЖНО:
  - Расширение НЕ защищает ключи в изображениях/видео
  - Расширение НЕ работает в PDF файлах
  - Всегда используйте переменные окружения (.env) для реальных ключей
  - Не полагайтесь только на расширение - это дополнительный уровень защиты

Минимальная версия:   Chrome 88+
Permissions:          storage, activeTab
Host Permissions:     <all_urls>
Размер:               ~15KB (без иконок)
Производительность:   < 10ms на сканирование страницы

Технологии:
  - TreeWalker API для обхода DOM
  - MutationObserver для динамических изменений
  - Clipboard API для копирования
  - chrome.storage.sync для синхронизации настроек
  - CSS filter: blur() для размытия
  - Regex паттерны для обнаружения ключей

Расширение не работает?
  → Перезагрузите страницу после установки расширения

Ключи не размываются?
  → Откройте popup и нажмите "Пересканировать страницу"
  → Убедитесь что защита включена (зеленый переключатель)

Иконки не отображаются?
  → Проверьте что файлы icon16.png, icon48.png, icon128.png в папке icons/
  → Они уже созданы и должны работать

Не работает на странице chrome://?
  → Это нормально, Chrome защищает служебные страницы от расширений

Расширение работает медленно?
  → Это маловероятно, но можно отключить MutationObserver в content-script.js
